# Translation Arbitrage — research log

**Thesis.** A *proven* PL↔ACT equivalence edge is a translation dictionary. Hunt for **asymmetries**: a result proved on one side with no counterpart on the other. Each asymmetry is a portable, falsifiable open problem — carry it across and you get genuine novelty; if the mirror already exists, the search just finds the paper.

**Method correction (codex, verified web search).** "Equivalence edge" is too loose as the unit. Replace it with a **preserved-structure checklist** before claiming a gap:
1. Objects / morphisms / judgments correspond (not just slogans).
2. The theorem's hypotheses are *expressible on both sides*.
3. The translation preserves the operations the *proof* uses (reindexing/substitution, quantifiers, fixpoints, monoidal product, order enrichment, modalities, operational adequacy).
4. The result is invariant under the equivalence, not presentation-specific.
5. The mirror side *has the required object* (separating conj needs BI/separation algebra; step-indexing needs guarded recursion/later; RG needs interference/stability structure).

**Two failure modes that kill a claimed gap.**
- *Already-known under another name* — "missing mirror" collapses to "exists as BI hyperdoctrine / separation algebra / Views / topos of trees, under a keyword I didn't search." (Same trap that sank the diss track, one level up.)
- *False transport* — the categorical abstraction forgets an operational hypothesis (faults, locality), so the ported theorem is silently false. The frame rule is the canonical example.

**Prior codex pass on the wp/Hoare edge (verified).** Frame rule, BI, higher-order frame rules, step-indexing, concurrent SL, persistent predicates — all already have categorical mirrors (BI hyperdoctrines, Views, topos of trees / synthetic guarded domain theory, Bizjak–Birkedal). This edge is *more* mined than first pitched. Survivors flagged "I did not find a mirror" (medium confidence): prophecy variables; ramification (RAMIFY); rely-guarantee as predicate transformers; full Iris specification logic in one categorical package.

## Convergent meta-finding (the actual deliverable)

All four branches independently land on the **same shape**, and it is stronger/more honest than the original "PL is decades ahead, just port it" pitch:

> Every mature PL↔ACT equivalence edge is **far more mined than it looks** — usually by a dedicated categorical sub-community (Birkedal school for SL; Fritz–Staton for probability; Katsumata–Orchard for grading). Naive "free theorem" arbitrage almost always hits *already-known-under-another-name*.
>
> The surviving asymmetries are **not unported theorems** — they are **un-assembled coherence theorems**. In each live case the PL side has already *isolated the exact algebraic structure* the loop/interference/future needs (laxly-iterable quantale; stabilisation-as-wp + rely-quotient-residuation; prophecy-erasure), and the categorical side has *all the component objects*, but **nobody has installed the structure into the canonical categorical framework and proved the coherence/precision theorem that makes it modular.** That theorem — not the translation — is the contribution.

This reframes the whole project a third time: **diss track → generous survey → "the yield of translation arbitrage is coherence theorems, not free theorems," with 4 worked, codex-verified examples ranked by how much assembly remains.** The honest, defensible, novel claim.

## Cycle 2 — H3 depth

### H3a: GMR parametrized-monad spoiler (opus, DOESN'T-KILL)

**Verdict:** GMR does **not** fill the graded-Elgot gap. Their dagger lives over Uustalu PARAMETRIZED monads (parameter = an *object* of the base category, discarded by the operator), which Orchard–Wadler–Eades prove is a structurally *distinct* gadget from a *monoid-graded* monad; the GMR dagger returns no index at all, so there is no `m*` grade-closure to extract.

**Claims:**
- GMR's parametrized monad is a bifunctor `⊞ : C × C → C` where `−⊞X` is a monad for each *object* `X ∈ C`. Their dagger takes `e : X → A⊞X` and returns `e† : X → A` — the parameter `X` is the recursion/coalgebra variable (resumptions over coproducts), and the output `e†` **discards it entirely**: a plain morphism in the single base category `C`, no external monoid/quantale, no grade tracked. [opus H3a, verified ar5iv 1603.02148 §def of parametrized monad, §complete Elgot ⊞-algebra, main thm 4.8 — "no external effect monoid or quantale indexing this construction… single base category C"]
- The decisive structural fact: graded ≠ parametrized. Orchard–Wadler–Eades, *Unifying Graded and Parameterised Monads* (arXiv:2001.10274, 2020) show they are **neither identical nor one a special case of the other** — both embed as distinct branches into a common apex, *category-graded monads* (lax functors `I^op → Endo(C)`). A monoid-graded monad indexes by an *element of an external pomonoid* `M` (lax monoidal `G : M → [C,C]`); a parametrized monad indexes by *pairs of objects* of a category `I` (`P : I^op × I → [C,C]`). Different indexing schemes; the "unify" is a coproduct-of-branches, not a subsumption. [opus H3a, verified ar5iv 2001.10274 Def 21 graded, Def 28 parametrized, Props 23/25/33]
- Even the unifying apex (category-graded monads) carries **no** iteration: OWE's text contains no Elgot/Kleene/dagger/closure operator. So the would-be bridge "instantiate GMR-Elgot with the effect-quantale graded Kleisli category and read off a grade-tracking dagger" has no published realization — the iteration operator is defined only on the *parametrized* branch (GMR), and that branch tracks no grade. [opus H3a, verified 2001.10274 §iteration: absent]
- Could you force the bridge by taking the effect-monoid `E` as a discrete/posetal category and forming a *category-graded* monad? That is exactly OWE's construction — but it lands on the **graded** branch, not the parametrized one GMR builds its dagger over (which needs an indiscrete/object-pair category). The two embeddings don't meet at the iteration operator. No free lunch. [opus H3a]

**The sharpened gap:** The grade-closure `m*` demanded by H3 has three properties GMR structurally cannot supply. (a) `m*` is a function of `m` **alone** — an element of the external quantale `E`, independent of objects `X,Y`; GMR's parameter is an *object of C*, and its dagger output carries *no* index. (b) `m*` must be the **least subidempotent grade ≥ (m ∨ unit)** (Gordon's laxly-iterable precision, TOPLAS 2021), an order-theoretic closure on `E`; GMR has no external ordered monoid in which to state subidempotency or least-ness. (c) The typing `f : X → T_m(Y+X) ⟹ f† : X → T_{m*}(Y)` must commute with the *graded* μ/η and subeffecting; GMR's `e† : X → A` commutes only with the parametrized (object-substitution) structure, which forgets the would-be grade at the unit. The gap is precisely: install an iteration operator on the **graded** branch of OWE's square (lax monoidal `E → [C,C]`) computing the quantale closure `m*`, with Conway/uniformity laws compatible with graded reindexing — a construction that exists on neither the graded branch nor (as a grade-tracker) the parametrized branch.

**Open questions:**
- Does OWE's category-graded apex admit *any* iteration operator, and if so does it specialise to grade-closure on the monoid branch or to GMR-style grade-forgetting resumption on the parametrized branch? (Decides whether the unification is a viable host for the construction or just a coincidence of indexing.)
- GMR's parameter is the recursion variable in `X → A⊞X`; in a graded reading the recursion variable and the effect grade are *different* axes. Can a *two-parameter* (Uustalu-parametrized **and** E-graded) monad host a dagger that iterates over the coproduct axis while accumulating `m*` on the grade axis? That bi-axial gadget, if it doesn't already exist, is the cleanest statement of the gap.

### H3c: Worked-example stress test (opus, CONDITIONAL — grade survives on laxly-iterable E, refining toward GRADE-TRACKS on the right quantales)

**Verdict:** The graded dagger is (c) **well-defined exactly on laxly-iterable E**, and on that class the grade *genuinely survives* iteration — it is NOT thrown away to top. The right picture: iteration is the grade-level closure `m ↦ m* = least subidempotent ≥ m ⊔ 1`, so the surviving loop-grade is the *image of that closure*, i.e. the sub-lattice of subidempotents. That image is rich (trace/security grades) for some quantales and coarse ({0,∞}) for others, but it is always the *correct* grade, never a discarded one. codex's "collapses to top" warning is real only for the degenerate quantales whose **only** subidempotents are {⊥,⊤}; that is a property of the specific E, not of the construction. The theorem has real content because perfectly reasonable gradings are **not** laxly iterable, and on those the dagger is undefined or non-canonical (Example 3).

Setup: graded monad `T` over ordered effect monoid `(E,·,1,≤)`; dagger `f : X → T_m(Y+X) ⟹ f† : X → T_{m*}(Y)` with `m* = ⋁_k m^k` when it exists (= least subidempotent ≥ m⊔1 — web-confirmed: Gordon TOPLAS'21, "least subidempotent effect ≥ x ⊔ I"; "every Kleene algebra corresponds to an iterable effect quantale"). Subidempotent: `e·e ≤ e`. Fixpoint law `f† = [η, f†] ∘ f` typechecks iff `m·m* = m*` (both sides at grade m*).

**The three+ worked examples:**

1. **Counting / writer, E = (ℕ∪{∞}, +, 0, ≤).** Complete quantale, + continuous over sups, ∞ absorbing. Subidempotents: `m+m ≤ m ⇔ m∈{0,∞}`. So `m* = ⋁_k k·m = 0` (m=0) else `∞` (m≥1) — exactly Gordon's least-subidempotent. Fixpoint typechecks: `m+m* = m*` (0+0=0; m+∞=∞). **Grade survives but coarsens to {0,∞}** = terminating-cost-free vs potentially-unbounded. The ∞ is the *honest* cost of an unbounded positive-cost loop, not "top thrown away." All five Gordon laws (extensive/idempotent/monotone/foldable/possibly-empty) hold on the 2-element subidempotent lattice. *Superficially looks like codex's collapse — but it is the correct, not the degenerate, reading.*

2. **Trace / language grading, E = (P(Σ*), concat-lift, {ε}, ⊆) — the free Kleene algebra.** Here `m·m = {uv} ≠ m` (m={a} ⟹ m²={aa}), so the grade is genuinely non-idempotent and `m* = ⋃_k m^k` = the *honest Kleene star* (m={a} ⟹ m*={ε,a,aa,…}), neither m nor ⊤=Σ*. Complete quantale; concat distributes over ⋃ ⟹ m* always exists, IS the least subidempotent, and the Conway/Kleene-algebra laws all hold (it *is* a Kleene algebra; web-confirms KA ⟹ iterable effect quantale). Graded dagger of a while-loop = its regular-language trace semantics. **Grade-tracks fully and nontrivially — m* is the regular star and is load-bearing.** Variant: **security/IFC lattice E = (L, ∨, ⊥)**, join-multiplication ⟹ every element idempotent ⟹ `m* = m`: a loop never escalates its label. m* intermediate, ≠⊤, grade preserved *exactly*. Two witnesses that the grade is not collapsed.

3. **Adversarial — m* absent / non-canonical (delimits the hypothesis).**
   - *3a, non-completeness.* E = (ℕ, +, 0, ≤) **without ∞** — a legitimate cost grading. For m≥1 the orbit {k·m} is unbounded with *no upper bound at all* in ℕ; the only subidempotent (0) is not ≥ m. So `m*` doesn't exist and `f†` is **undefined for every body with m≥1**. Not laxly iterable. The single missing element (∞) is exactly what makes Example 1 work.
   - *3b, no LEAST subidempotent (sharp case).* E = chain `1<m<m²<…` adjoined with two incomparable maximal idempotents `a,b`, each above the whole chain (`mⁱ<a`, `mⁱ<b`), `a∥b`, `a·a=a`, `b·b=b`, both absorbing. Chain elements aren't subidempotent (`mⁱ·mⁱ=m^{2i}>mⁱ`); subidempotents ≥ m are exactly `{a,b}`, **incomparable, no least element** — precisely the web-confirmed failure of laxly-iterable ("set is empty or has a least element"). The orbit ⋁mⁱ has two incomparable minimal upper bounds, so `m*` is **non-canonical**: dagger lands in grade a *or* b with no principled choice ("diverges via channel A" vs "B"). Two distinct loop semantics, no canonical one. Cleanest proof that `(-)*` is a *partial* operator and laxly-iterability is doing real work.

**Preserved-structure checklist:** *Result invariant?* Yes — m* is order-theoretic (least subidempotent), iso-invariant; "m* undefined/non-least" (3a,3b) is equally structural. *Hypotheses expressible both sides?* "Laxly iterable" (Gordon) ⟷ "`⋁_k m^k` exists and is subidempotent" — categorically, a partial closure/idempotent-monad on the grading object compatible with the lax functor. Ex 1,2 satisfy it; 3a,3b are explicit counter-instances — exactly the missing categorical hypothesis H3/H3a flagged. *Proof uses structure the example has?* Fixpoint needs `m·m*=m*` (subidempotency + 1≤m*); holds in 1,2; *fails to typecheck* in 3 (m* absent/ambiguous). False-transport bites precisely at non-laxly-iterable E — confirming H3's "false-transport crux."

**Does the grade survive iteration, or collapse to top?:** It survives, with a precise caveat. Iteration is a **closure onto the subidempotent sub-lattice** of E; the grade is never silently discarded, it is *projected* onto `Im((-)*)`. On Kleene-algebra / join-semilattice gradings (Ex 2) that projection is the honest star — full grade-tracking, m* genuinely needed. On {0,∞}-type cost monoids (Ex 1) it is coarse but still correct (∞ = real unbounded cost). The *only* regime matching codex's "ungraded-Elgot-in-disguise" fear is the degenerate sub-case `Im((-)*) = {⊥,⊤}`, a property of the specific quantale, not the construction. Honest headline: **graded dagger = ungraded Elgot landing in the idempotent-closure of the grade, well-defined iff E is laxly iterable.**

**Open questions:**
- Is `Im((-)*)` (subidempotents of E) always a *retract* of E in the grading category, with the dagger factoring through the corresponding idempotent comonad? If so, "graded Elgot monad" = "Elgot monad on the subidempotent-graded subcategory" — possibly the whole theorem, and it dovetails with H3a's "install iteration on the graded branch."
- For 3b, do the two completions (a vs b) correspond to two distinct *lax* extensions of the functor — i.e. is non-canonicity of m* the exact obstruction to a *unique* graded dagger, making laxly-iterable both necessary and sufficient for canonicity?
- Does guardedness rescue 3a? An ω-continuity/guarded condition forcing the missing sup to exist in the ideal-completion of E — is the right theorem "complete-Elgot on the ideal-completion of E"?

### H3b: Construction of the graded Elgot monad (opus, COHERENT)

**Verdict:** A clean nondegenerate definition exists. The categorical coherence laws *reproduce* Gordon's laxly-iterable conditions almost exactly (fixpoint ⟹ unfold + unit-below; codiagonal ⟹ subidempotent; naturality ⟹ monotone), *exceed* them on one point (dinaturality predicts a grade-level **sliding law** Gordon's single-body closure never states), and *fall short* on one (maximal-precision/leastness is an independent universal property, not forced by fixpoint+codiagonal+naturality). The feared collapse (m* = ⊤ trivializes the grade) does **not** happen: the grade survives on the entire non-looping fragment and on pure loops; only iterating an effectful body honestly costs ⊤, which is information, not erasure. (Convergent with H3c via a different route: H3c reaches "closure onto the subidempotent sub-lattice"; this entry derives *which law forces which* condition and exhibits the model.)

**The definition.** Fix an ordered monoid (E, ·, 1, ≤) with a partial unary closure (-)* (Gordon). A graded monad T : E → [C,C] over a base where the relevant iteration exists (C ω-cpo-enriched, or T_1 = a base Elgot monad B on C) is a **graded Elgot monad** if it is equipped with a family
> (-)†_m : C(X, T_m(Y+X)) → C(X, T_{m*}(Y))   (defined on the domain of (-)*)

natural in X, satisfying (write ι_{a,b} : T_a → T_b for the subeffecting coercion when a ≤ b; • for graded Kleisli composition g • f = μ_{a,b} ∘ T_a(g) ∘ f : X → T_{a·b}):

- **Graded fixpoint.** f† = ι_{m·m*, m*} ∘ ([ι_{1,m*}∘η_Y , f†] • f). Grade audit: f : X → T_m(Y+X); the cotuple [ι∘η_Y, f†] : Y+X → T_{m*}Y (needs **1 ≤ m\*** to lift η); Kleisli-compose with f gives grade **m·m\***; the equation lands in T_{m*}Y only after coercing, which needs **m·m\* ≤ m\***. So the single law forces *both* of Gordon's unfold inequalities. (m ≤ m* is then *derived*: m = m·1 ≤ m·m* ≤ m*.)
- **Naturality in Y.** For pure h : Y → Y′, (T_m(h+X)∘f)† = T_{m*}(h)∘f†. Grade-flat (no new E-condition). For Kleisli g : Y → T_k Z, the Kleisli-naturality form pushes a k onto both sides equally; monotonicity of the coercions forces **m ≤ n ⟹ m\* ≤ n\*** (Gordon monotone).
- **Dinaturality (Bekič/sliding).** For f : X → T_m(Y+Z), g : Z → T_n(Y+X), fusing the alternating loop the two ways forces, at grade level, **m·(n·m)\* ≤ (m·n)\*·m** — Kleene's *sliding* rule. Gordon's one-body closure never exposes this; the two-grade categorical loop does. (Holds in the model below: both sides = ∞ once a body is non-pure.)
- **Codiagonal.** For h : X → T_m((Y+X)+X), iterating the inner X then the outer X vs. merging first: double dagger has grade (m*)* = **m\*\***, single (merged) dagger has grade **m\***. The law equates them ⟹ **m\*\* = m\***, i.e. m* **subidempotent** (m*·m* ≤ m*). Exactly Gordon's subidempotency, *forced by codiagonal alone*.
- **Uniformity.** Standard quotient/coalgebra-map condition on X; grade-flat.
- **μ/η/⊑ compatibility.** (1)* = 1 (pure loop stays pure); η of a grade-1-coerced map iterates to η; the dagger commutes with graded μ and with subeffecting ι. This is the *precision/coherence* content cycle-1 H3 asked for — it constrains T, not E, so adds no E-condition (it is the modularity the categorical packaging buys over Gordon's grade-only analysis).

**Which laws force which grade conditions (match / exceed / shortfall vs Gordon laxly-iterable):**

| Gordon condition | Forced by | Status |
|---|---|---|
| 1 ≤ m* | graded fixpoint (lift η into cotuple) | MATCH |
| m·m* ≤ m* (unfold) | graded fixpoint (land the composite) | MATCH |
| m ≤ m* | *derived* from the two above | MATCH (free) |
| m*·m* ≤ m* (subidempotent, m**=m*) | **codiagonal** | MATCH |
| m ≤ n ⟹ m* ≤ n* (monotone) | naturality + coercion monotonicity | MATCH |
| m·(n·m)* ≤ (m·n)*·m (sliding) | **dinaturality** | EXCEED (Gordon's single-body closure doesn't state it) |
| m* is the *least* such (maximal precision) | — | SHORTFALL: independent universal property (initiality among valid target grades); fixpoint+codiagonal+naturality give *a* subidempotent closure, not *the least* one without an added leastness clause |

So the thesis "the categorical coherence laws reproduce Gordon's laxly-iterable conditions" is **confirmed**, with two refinements: the categorical theory *predicts an extra law* (sliding) and *misses one* (leastness must be added as a universal property — fine, since leastness is what makes Gordon's operator *canonical*, and categorically it's just "the dagger factors through every other valid target grade"). This pairs with H3c's example 3b: non-canonicity of m* (two incomparable minimal subidempotents) is exactly the failure of the leastness clause.

**Would-be main theorem (falsifiable).** Let (E,·,1,≤,*) be laxly-iterable: (-)* a partial closure with 1≤m*, m·m*≤m*, m*·m*≤m*, monotone, and m* least such. Let C be an Elgot base (ω-cpo-enriched, or carrying base Elgot monad B) and T : E → [C,C] a graded monad whose coercions and each T_m preserve the colimits B's iteration uses. **Then** T carries a unique grade-tracking dagger (-)†_m : C(X,T_m(Y+X)) → C(X,T_{m*}Y) satisfying graded {fixpoint, naturality, dinaturality, codiagonal, uniformity}; conditions (i)–(iv) on E are *not assumed independently* but *forced* by those laws as tabulated; and the Grothendieck total category ∫T (resp. the colimit T_⊤ = colim_m T_m when ⊤ exists) carries an ordinary Elgot monad whose dagger is the grade-image of (-)†. Falsifiers: (F1) drop subidempotency from E ⟹ codiagonal *must* fail on a nested loop with m**>m*; (F2) the graded writer below is a model with non-trivial grade off the looping fragment; (F3) E without the needed suprema (e.g. (ℕ,+) sans ∞) ⟹ dagger *necessarily partial* — Gordon's *partial* closure (= H3c example 3a).

**Nondegenerate model (refutes the collapse).** Take E = (ℕ∪{∞}, +, 0, ≤). m* must satisfy 0 ≤ m*, m+m* ≤ m*, 2m* ≤ m*. In (ℕ∪{∞},+): 2x ≤ x ⟹ x∈{0,∞}; and m+x ≤ x with m≥1 ⟹ x=∞. Hence **0\* = 0, m\* = ∞ for m ≥ 1.** Model: graded writer over a base Elgot monad B on C (ω-cpo, or B = maybe on **Set**), output monoid W,
> T_n X = B(X × Wⁿ),  η = η^B into T_0 X = BX,  μ_{m,n} concatenates write-blocks via B's strength.
Dagger: f : X → T_m(Y+X) = B((Y+X)×Wᵐ). m=0: f† is *B's own* Elgot dagger (pure tail recursion, no writes) into T_0Y = BY. m≥1: B-iterate accumulating per-step write-blocks; a non-terminating run accrues an infinite write-stream into T_∞Y = B(Y × W^∞). Every grade equation checks: m·m*≤m* (m≥1: m+∞=∞✓; m=0: 0✓), subidempotent (∞,0 ✓), monotone ✓, sliding (both sides ∞ once a body is non-pure ✓), and crucially **1\*=1** i.e. 0\*=0: a loop whose body writes nothing but may diverge keeps write-grade 0 — the divergence lives in B, not the count. ### H3: Graded monads ↔ graded effect systems (opus, PARTIAL — core saturated, iteration corner ALIVE)

**Verdict:** The core dictionary is thoroughly mined in BOTH directions — this edge is *more* saturated than wp/Hoare. The one surviving asymmetry is **iteration**: PL has a developed theory of which effect-grading structures admit a canonical loop/iteration operator *on the grade* (Gordon's iterable effect quantales), while the categorical iteration theory (Elgot/Kleene/while monads) is **ungraded** — it forgets the grade. No "graded Elgot monad / graded Kleene monad / graded while-monad" exists; Gordon himself flags the categorical generalization as open.

**Baseline already-translated (do NOT claim these):**
- Katsumata, "Parametric Effect Monads and Semantics of Effect Systems" (POPL 2014), http://bibtex.github.io/POPL-2014-Katsumata.html — the foundational dictionary: graded (parametric effect) monad = lax monoidal functor from a preordered-monoid (monoidal) category of effects into [C,C]; ports Kleisli triples, state/continuation monads, Plotkin-Power algebraic operations, ⊤⊤-lifting. The categorical characterization the task flagged (graded monad = lax monoidal functor on the effect monoid) is *exactly* this paper.
- Wadler-Thiemann, "The Marriage of Effects and Monads" (TOCL 2003), https://www.cs.tufts.edu/~nr/cs257/archive/phil-wadler/effectstocl.pdf — effect system ↔ family of monads over a bounded semilattice; Orchard-Petricek-Mycroft "The semantic marriage of monads and effects" (arXiv:1401.5391) is the semantic version.
- Gaboardi, Katsumata, Orchard, Sato, "Graded Hoare Logic and its Categorical Semantics" (ESOP 2021), https://arxiv.org/abs/2007.11235 — graded analogue of the wp/Hoare story (cost, probability, differential privacy as gradings). Overlaps the wp edge: graded Hoare is done.
- EM/Kleisli/algebraic-presentation story is built out: Fujii-Katsumata "Towards a Formal Theory of Graded Monads"; Katsumata-McDermott-Uustalu-Wu "Flexible Presentations of Graded Monads" (ICFP 2022), https://dl.acm.org/doi/10.1145/3547654; McDermott-Uustalu "Flexibly Graded Monads and Graded Algebras" + "Canonical Gradings of Monads" (arXiv:2307.16558) — graded monad = relative monad; graded algebras, presentations, locally-graded categories.
- Graded monad *transformers* and **graded distributive laws** exist (Uustalu, "Grading monads, comonads, distributive laws", CLAP Scotland; "Parametric Distributive Laws", arXiv:2509.22208) — the task's distributive-law/transformer candidate is NOT a gap.
- Effect **handlers** ↔ grading is done: "Category-Graded Algebraic Theories and Effect Handlers" (CatEff), arXiv:2212.07015 — handlers as implementation of category-graded effects, with soundness + adequacy. Not a gap.

**Structure checklist on the survivor (graded iteration):**
1. *Objects correspond?* Effect-monoid grade ↔ grading monoidal category: yes (Katsumata). But iterated grade n* needs the grade object to carry a Kleene-star/closure; the grading category has no such star in current axioms. Partial — the porting target.
2. *Hypothesis expressible both ways?* PL hypothesis = "laxly iterable" (iteration map is a partial closure operator on the quantale). Categorical mirror = a closure / idempotent-monad structure on the grading category compatible with the lax monoidal functor. Expressible in principle, ABSENT in practice — the live gap.
3. *Proof operations preserved?* Ungraded iteration uses guardedness / ω-continuity / Conway laws (fixpoint, naturality, codiagonal, uniformity). Open whether these even *type-check* under grading: iterating f : X → M_n(Y+X) must land in a well-defined grade M_{n*}Y, and codiagonal/uniformity must respect the grade-star. Exactly where **false transport** could bite (the categorical f† may not have a well-defined grade), so the claim is research-grade, not transcription.
5. *Mirror side has the object?* No — graded monads don't currently package an iteration operator on the grade. Gap survives.

**Ranked falsifiable gaps:**
1. (strongest) *No graded Elgot/Kleene/while monad.* No categorical structure = a graded monad with a coherent iteration operator producing the Kleene-starred grade (Conway/uniformity/codiagonal laws compatible with reindexing), plus a Goncharov-style characterization (graded analogue of "ω-continuous ⇒ complete Elgot" or the while-monad axioms) and the "laxly-iterable" hypothesis recast as a categorical closure on the grading category.
