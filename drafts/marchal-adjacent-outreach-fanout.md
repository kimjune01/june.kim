## THE THROUGHLINE (codex-validated)

The filter that separates a real target from a thematic neighbor:

> **Does their work convert model reasoning from persuasive text into externally checkable obligations
> — claims, evidence links, failure conditions, replayable tests that survive distrust of the model?**
> If yes → mechanism twin. If it only studies whether reasoning is *faithful / calibrated / virtuous*
> → adjacent vocabulary, email as *contrast*, not kinship. If normative/governance → skip.

(lab-cluster + adjacent-academics). The debate / prover-verifier / AI-control family is the densest vein.
