I accepted your suggestion and put a plain-language summary on pageleft.cc answering your two questions. 

Pageleft = PageRank + Copyleft

It is a search index of web pages under share-alike licenses. A query returns passages with source URLs and licenses attached, for a human or a coding agent to quote or adapt under the same license. gnu.org and fsf.org were not indexed here because they are CC BY-ND, not BY-SA. ND forbids building on top, so the index excludes them by design.

As for its motivation, ideas have become hard to find. Advertising and marketing copy dominate the searchable web, crowding out everything else. Two people working on the same question in different vocabularies have a harder time finding each other than they did a hundred years ago. Copyleft filters by authorial intent; embeddings, not keywords, make ideas adjacent in search.

Onto the principles that matter when writing software costs almost nothing:

Until now, copyleft on prose and copyleft on code did different things. A share-alike essay propagated information; a share-alike program propagated a runnable tool. The essay could influence later writing; it could not become software.

That has changed. Coding agents can now turn sufficiently precise prose into working programs. An essay that specifies a behavior compiles to code, and if the essay is under a share-alike license, the program inherits share-alike obligations. Copyleft is irrevocable; derivatives inherit the obligation; the commons cannot shrink.

A corpus of copylefted prose is now a source of free software. Each essay added lowers the cost of producing free software, because a coding agent can now compose from it rather than write from scratch. At a threshold, producing free software becomes cheaper than paying for proprietary alternatives; past that point, the rational choice is copyleft, and the commons wins by default, instead of by persuasion. PageLeft makes that equation trivial. Without an index, "compose from prior art" is a nice idea and a linked list of attributions; with one, it is a curl and a lattice.

Aaron Swartz argued that locked-up knowledge was a moral failure and that sharing was a duty. He was right about the duty, but without mechanism. Coding agents reading copylefted prose are that mechanism.

Open beat closed platforms in every prior round: TCP/IP over proprietary networks, SMTP over CompuServe, HTTP over AOL. The same pattern applies above code. PageLeft stands on the side free software has held for forty years. 

If I write software for myself and attach a free license without sharing it, the license is meaningless. Without recipients, no rights are exercised. Free software in a closet is free by license and nothing more. 

What is the maximal opposite of closeted software? Work published on the open web, indexed and ranked, without gatekeeping. Without an index, free software is a pile of programs found by luck. With one, the works compose: each contribution lowers the cost of the next. The four freedoms protect each program. An index makes it a commons.

Gatekeeping does not scale, and it does not have to. PageLeft uses the license as its gate: any page under a share-alike license qualifies. If someone disagrees with the filter, they can fork the code and run their own index with their own criteria. For free software to become the default rather than a principled minority, we must be united and the work must be searchable without anyone's permission.
