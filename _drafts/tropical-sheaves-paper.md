# Tropical Sheaves on Temporal Event Graphs: An Obstruction and a Worked Example

## Abstract

Temporal graph theory, sheaf-theoretic network topology, and predictive coding / codec engineering all study directed dependency structures with time-ordered composition, but they do so with largely disjoint vocabularies. This paper gives a concrete crosswalk through a shared object: the temporal event graph, a static DAG whose vertices are events and whose edges encode temporal admissibility. We make three claims. First, a naive tropicalization of sheaf-theoretic max-flow/min-cut on event graphs fails: tropical global sections encode earliest-arrival potentials, not conserved commodities. We give a counterexample, prove that no purely local edge-based cut notion can restore duality, and show that a global potential barrier does restore it. Second, on prediction chains and trees with drift, the relevant checkpoint-spacing quantity is the expected irreducible residual (not cohomological dimension, which increases with checkpoints), yielding an optimal spacing of L* ≈ √(2λ/(kσ²d)) where k is the branching factor. Third, we state a precise open rate-distortion problem for journey observables on temporal graphs.

## 1. Introduction

Three communities study the same broad combinatorial object.

Temporal graph theory studies time-respecting paths, or journeys, on edge-time data. The central difficulty is that reachability is constrained by temporal order, so static graph intuitions fail. Event-graph constructions convert a temporal graph into a static DAG that encodes admissible compositions of events [Casteigts et al., 2012; Mellor, 2018; Kivelä et al., 2018].

Sheaf-theoretic network topology studies local-to-global consistency on directed cell complexes. In this language, a prediction rule is a restriction map, a consistent assignment is a global section, and failure of globalization is measured cohomologically [Krishnan, 2014; Ghrist and Hiraoka; Hansen and Ghrist, 2019; Seely, 2025].

Codec and predictive-coding engineering studies prediction dependency DAGs, clamped reference frames, and rate-distortion tradeoffs on chains of predictions [Chou and Miao, 2006; Sullivan and Wiegand, 1998].

The contribution is a crosswalk, not a unification claim. The temporal event graph is a common static DAG on which techniques from the other two fields can be tested directly. One transfer fails instructively: naive tropical sheaf MFMC does not survive on event graphs. Another succeeds modestly: relative-sheaf calculations on chains recover a checkpoint-spacing law familiar from codec practice.

## 2. Preliminaries

A temporal graph is a vertex set together with timestamped interactions. We use the TVG viewpoint of Casteigts et al. [2012]: a journey is a path whose edge times are nondecreasing and respect travel or waiting constraints. Mellor's temporal event graph (TEG) replaces each timestamped interaction by a vertex and adds a directed edge when one event can follow another in time [Mellor, 2018]. Kivelä et al. refine this into weighted event graphs that encode waiting-time constraints and capture all admissible time-respecting paths under a threshold parameter [Kivelä et al., 2018]. In all cases relevant here, the event graph is a static DAG.

A cellular sheaf on a directed graph assigns a stalk to each vertex and edge and restriction maps from incident vertices to the edge stalk. For a graph sheaf, a global section is an assignment to vertex stalks whose images agree on every edge. Over rings, cochains and coboundaries express prediction error and conservation constraints. Krishnan [2014] gives a semiring-valued MFMC formalism on digraphs; Ghrist and collaborators study sheaf-cohomological flow/cut dualities in additive settings.

The tropical semiring is

T = (R≥0 ∪ {∞}, ⊕, ⊗) = (R≥0 ∪ {∞}, min, +).

Here tropical addition chooses the earlier of two arrival times, and tropical multiplication accumulates delay. This makes it a natural coefficient system for earliest-arrival scheduling. T has no additive inverses, so "difference of inflow and outflow" has no direct tropical analogue.

## 3. Crosswalk Setup

The shared object is a static DAG G_E, viewed in three ways.

From temporal graph theory, G_E is an event graph: vertices are events, edges are admissible temporal concatenations.

From coding theory, G_E is a prediction dependency DAG: if event e' depends on event e, then decoding or prediction must respect that edge [Chou and Miao, 2006].

From sheaf theory, G_E is a cell complex on which local prediction rules can be placed as restriction maps [Seely, 2025].

Sections 4 and 5 test two transfers: one fails, one succeeds.

## 4. Main Result: Naive Tropical MFMC Breaks on Event Graphs

### 4.1 Sheaf Construction

Let G=(V,E) be a directed event graph with delay label w:E→R>0. Define a sheaf F_w on G by

F_w(v) = T, F_w(e) = T

for every vertex v and edge e:u→v, with restriction maps

ρ_{e←u}(x) = x + w(e), ρ_{e←v}(y) = y.

A global section is thus an assignment x_v ∈ T such that for every edge e:u→v,

x_v = x_u + w(e).

Global sections are consistent schedules: the value at a node is the arrival time forced by upstream timing. Note that at merge vertices (multiple incoming edges), the sheaf condition requires all incoming path lengths to agree. This is stricter than the tropical shortest-path recursion x_v = min_u(x_u + w(u,v)), and global sections may not exist on generic weighted DAGs. When they do exist (e.g., on paths, or on DAGs where all directed paths to each vertex have equal total delay), the section value at the sink equals the common path length.

The naive tropical "cut capacity" of an s-t cut C ⊆ E is

κ_naive(C) = ⊕_{e∈C} w(e) = min_{e∈C} w(e).

This is the direct tropical analogue of summing capacities across a cut.

### 4.2 Four-Vertex Counterexample

Consider the diamond DAG with vertices s, a, b, t and edges

s→a, s→b, a→t, b→t,

all of delay 1. This can be realized as a TEG subgraph coming from two simultaneous first events and two simultaneous second events.

Write the edge stalk conditions explicitly:

x_a = x_s + 1, x_b = x_s + 1,
x_t = x_a + 1, x_t = x_b + 1.

Anchor the source at x_s = 0. Then

x_a = 1, x_b = 1, x_t = 2.

Hence the anchored global section is unique and equals

(x_s, x_a, x_b, x_t) = (0, 1, 1, 2).

Now compute the naive cut capacity. For the cut separating {s} from {a, b, t}, the cut set is

C = {s→a, s→b}.

Therefore

κ_naive(C) = min{1, 1} = 1.

But the sink value of the global section is

x_t = 2.

So the max-flow/min-cut style inequality

flow value ≤ cut capacity

would read

2 ≤ 1,

which is false.

This is not a numerical accident. Section value accumulates delay along a path (+). Cut value selects the cheapest crossing edge (min). These are different operations with no reason to be dual.

### 4.3 General Obstruction

**Theorem 4.1.** Let P be a directed path v_0 → v_1 → ... → v_k with k ≥ 2 and Σ_{i=2}^k w_i > 0. Equip P with the tropical sheaf F_w and anchor x_{v_0} = 0. Then for the source cut C = {v_0 → v_1},

κ_naive(C) = w_1 < Σ_{i=1}^k w_i = x_{v_k}.

**Proof.** Edge equations force x_{v_i} = x_{v_{i-1}} + w_i, so x_{v_k} = Σ w_i. Since Σ_{i=2}^k w_i > 0, we have w_1 < Σ w_i. ∎

**Corollary 4.2.** Any temporal event graph containing a multihop journey with positive total delay beyond the first edge contains a subgraph on which naive tropical cut capacity fails to upper-bound section value.

### 4.4 Impossibility of Local Cuts

No purely local edge-based cut notion can restore duality.

**Proposition 4.3.** Let κ: (E, w) → R be any cut functional depending only on the weights of edges crossing the cut. Then for any finite value κ assigns, there exists a tropical sheaf on an event graph with the same cut-edge weights whose section value exceeds κ. Hence no finite-valued local cut functional can upper-bound section value in general.

**Proof.** Take the diamond DAG (s→a, s→b, a→t, b→t). Fix cut C = {s→a, s→b} with weights w(s→a) = w(s→b) = 1. Any local κ assigns a fixed finite value κ(C) depending only on these weights. Now set the non-cut edges a→t and b→t to delay M for arbitrary M > 0. The sheaf condition forces x_t = 1 + M. Choose M > κ(C). Then x_t > κ(C). Since M is unbounded, no finite κ(C) suffices. ∎

### 4.5 What Does Work: The Potential Barrier

Exact duality exists, but the dual object is global, not local.

For any edge cut F separating s from t, define the distance-decorated cut capacity:

κ_pot(F) = min over (u,v) ∈ F of [d(s,u) + w(u,v) + d(v,t)]

where d(·,·) is the shortest-path distance in the tropical sense (sum of delays along the shortest path).

**Observation 4.4.** κ_pot(F) = d(s,t) for every s-t separator F. Every s-t path crosses F, so d(s,t) ≤ d(s,u) + w(u,v) + d(v,t) for each cut edge, and a shortest path crosses some cut edge with equality.

This is a corollary of shortest-path structure. Restoring duality requires global distance information from both sides of the cut, confirming that no local functional suffices.

### 4.6 Why the Obstruction Is Structural

Tropical sections are potentials: they encode earliest-arrival schedules. Classical flows are conserved commodities. Time accumulates along paths (tropical multiplication = addition of delays) but selects across paths (tropical addition = min of arrival times). MFMC duality requires these two operations to be dual. In the tropical semiring, they are not: + and min play fundamentally different structural roles. The obstruction is algebraic (no additive inverses), not combinatorial.

## 5. Worked Example: Checkpoint Spacing on Linear Prediction Chains

Consider a chain v_0 → v_1 → ... → v_n with d-dimensional stalks and linear restriction maps W_i: R^d → R^d. Clamp the endpoints v_0 and v_n. This is the sheaf-theoretic model of a prediction chain with fixed source and target states.

Two facts are standard in relative sheaf cohomology [Hansen and Ghrist, 2019; Seely, 2025]. First,

dim H^1_rel = d

for an endpoint-clamped chain, independent of n. Second, inserting c interior checkpoints decomposes the chain into c+1 clamped segments and yields

dim H^1_rel = (c+1)d.

So checkpoints increase H^1; they do not remove it.

Checkpointing is not topologically beneficial. The operational benefit appears only after adding drift.

Assume now a homogeneous predictor W_i = W and Gaussian drift

x_{i+1} = W x_i + ε_i, ε_i ~ N(0, σ²I_d).

Let r* denote the irreducible residual obtained by the relative least-squares solve on a segment of length n. Then the standard Gramian computation gives

E[‖r*‖²] = σ² tr G_n(W), G_n(W) = Σ_{k=0}^{n-1} (W^T W)^k.

If μ_1, ..., μ_d are the eigenvalues of W^T W, then

tr G_n(W) = Σ_{i=1}^d (1 - μ_i^n)/(1 - μ_i),

with the usual limit n when μ_i = 1.

Residual growth depends on squared singular values near or above 1, not on the condition number. A badly conditioned but contractive predictor accumulates little residual; a well-conditioned near-unitary predictor can accumulate much.

Now impose a checkpoint cost λ per inserted checkpoint and restrict to the nearly-unitary regime μ_i ≈ 1, where

tr G_L(W) ≈ dL

for a segment length L. The average cost per unit time is then approximated by

J(L) ≈ λ/L + σ²d/2 · L.

Minimizing yields

L* ~ √(2λ/(σ²d)).

This recovers the usual codec heuristic: more drift (σ² larger) implies shorter GOPs; more expensive checkpoints (λ larger) implies longer GOPs. The square-root law itself is known from HPC checkpointing [Young, 1974]. The contribution is the interpretation via sheaf cohomology and the following extension.

### 5.1 Fan-out: hierarchical prediction structures

For a tree-structured prediction DAG with branching factor k (one reference frame used by k downstream predicted frames), the total residual over all leaves scales as k · σ² · tr G_L(W). The checkpoint cost is amortized across k descendants, giving

J(L) ≈ λ/L + kσ²d/2 · L.

Minimizing:

L*_fanout ≈ √(2λ/(kσ²d)).

The 1/√k factor means fan-out makes shared checkpoints more cost-effective. One root checkpoint protects all k descendants. This is consistent with the hierarchical prediction structures used in H.264/HEVC, where shared references are refreshed more aggressively than leaf frames.

For trees, per-leaf residual depends only on depth, not topology. No cross-terms appear because each ancestor has a unique path to each descendant. Cross-terms appear only at DAG reconvergence points, where multiple paths from the same noise source create interference via ‖Σ_p A_p‖²_F. Practical B-frame structures involve such reconvergence; the tree model is a simplification.

Searching the H.264, HEVC, and VVC overview papers [Sullivan et al.; Schwarz, Marpe, Wiegand 2006; Bross et al. 2021], hierarchical B-frame analysis literature, and adaptive GOP optimization papers across Google Scholar and Semantic Scholar, we found no closed-form derivation of checkpoint spacing as a function of branching factor. Codec engineers tune hierarchical GOP structure empirically.

## 6. Open Problem: Rate-Distortion Bounds for Journey Observables

We state one precise version.

Fix a vertex set V and horizon T. Let X_T be a random temporal graph (or link stream) on [0,T]. For u,v ∈ V and start time t, let τ_X(u,v;t) ∈ [0,T] ∪ {∞} be the foremost-arrival time from u to v departing no earlier than t. Define distortion

d_η(X, X̂) = (1/(|V|(|V|-1)T)) Σ_{u≠v} Σ_{t=0}^{T-1} 1[|τ_X(u,v;t) - τ_{X̂}(u,v;t)| > η or exactly one is ∞].

Define the operational rate-distortion function

R(D) = limsup_{T→∞} inf_{E[d_η(X_T, X̂_T)] ≤ D} (1/T) log M_T,

where M_T is the encoder codebook size.

The problem is to determine or sharply bound R(D) for natural source classes.

Achievability: timestamp coarsening into bins of width Δ gives zero distortion whenever Δ ≤ η, at rate strictly below lossless. This is scalar quantization on temporal structure.

Converse: the natural sufficient statistic is the interval transfer tensor (pairwise foremost-arrival times within each checkpoint interval). These are i.i.d. across intervals, supporting a blockwise lower bound. But journeys cross interval boundaries and share pivotal events, creating a coupling gap that prevents a tight end-to-end converse.

Related work: Lamarche-Perrin [2019] optimizes event-level distortion on link streams; Bilò et al. [2022] control stretch combinatorially. Neither addresses journey-aware R-D bounds.

## 7. Discussion

The crosswalk reveals a vocabulary barrier more than a mathematical one.

Temporal graph theory already has the right object: the event graph. Sheaf theory already has the right language for local consistency and irreducible error. Coding theory already has the right optimization instinct: pay bits where failure propagates.

Section 4 prevents a category error: tropical sheaf sections are consistent schedules, not conserved flows. No local edge-based cut can bound them, but a global potential barrier restores duality as a corollary of shortest-path structure. Section 5 shows the opposite: on prediction trees with drift, the sheaf language recovers codec practice and extends it with a fan-out scaling consistent with hierarchical GOP design.

The likely path forward is selective transfer, not wholesale unification. Some notions match exactly; others only after reformulation. Making those boundaries explicit is the main purpose of the paper.
